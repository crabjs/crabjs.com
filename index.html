The requirements of Crab.js

The server part of crab requires a server (duh) that can run Node.js v6.x.

For the client part, crab does’t aim to work for everyone.

In order to do what it’s created to do, and do it best, crab embraces some of the newest web technologies
– those only supported in modern browsers.

"How to check if my browser is, uh, modern enough?" you ask.
Well, if you’re not using a modern browser, you wouldn’t be here at first place
this website doesn’t look broken to you, chances are it is.

The installation and documentation of Crab.js

git clone https://github.com/crabjs/crabjs-cms

cd crabjs-cms

npm install

npm server

Server started on http://localhost:3000/

For more installation guides, documentation, and troubleshooting, head over to the repo’s Documentation and Releases.
